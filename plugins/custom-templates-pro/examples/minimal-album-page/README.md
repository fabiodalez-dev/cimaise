# Minimal Album Page Template

Template completo per pagina album con design minimalista e pulito.

## Caratteristiche

- ✅ Design minimale e pulito
- ✅ Tipografia elegante
- ✅ Header centrato con categorie
- ✅ Sezione equipment organizzata
- ✅ Griglia 3 colonne (responsive)
- ✅ Hover con zoom immagini
- ✅ Dark mode automatico
- ✅ Integrazione PhotoSwipe

## Cosa Include

Questo template include la **pagina album completa**:
- Header con titolo, categorie, data e location
- Sezione descrizione (body)
- Sezione equipment (fotocamere, obiettivi, pellicole)
- Galleria immagini integrata

## Installazione

1. Comprimi questa directory in un file ZIP
2. Carica tramite Admin → Custom Templates → Carica Template
3. Seleziona "Template Pagina Album Completa"
4. Upload completo!

## Personalizzazione

Puoi modificare:
- **Colori**: Modifica variabili colore in styles.css
- **Tipografia**: Cambia font-family
- **Layout griglia**: Modifica grid-template-columns
- **Spaziatura**: Adatta padding e margin

## Note

Perfetto per portfolio fotografici professionali che vogliono un look pulito e senza distrazioni.
